# PWA Icons

This directory should contain the following icon files for the PWA manifest:

- icon-72x72.png
- icon-96x96.png  
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- search-96x96.png
- mood-96x96.png
- close-96x96.png

You can generate these icons using:
1. Online tools like favicon.io
2. Design software like Figma, Sketch
3. Command line tools like ImageMagick

The icons should follow these guidelines:
- Use the brand colors (#667eea, #10b981, #ef4444)
- Keep them simple and recognizable
- Ensure good contrast on both light and dark backgrounds
- Test on different device sizes
